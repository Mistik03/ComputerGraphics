Each practical is organized in its own folder.

---

## How to Run

### Prerequisites
- [Node.js](https://nodejs.org/) (includes npm)

---

### Steps

1. **Clone this repository**
   ```bash
   git clone https://github.com/Mistik03/ComputerGraphics.git
   ```
   
2. **Navigate into the practical folder you want to run (example):**
   ```bash
   cd .../ComputerGraphics/PracticalOne
   ```
   
3. **Install dependencies (this will automatically install Vite, Three.js, and any other libraries the project needs)**
   ```bash
   npm install
   ```
   
4. **Start the server**
   ```bash
   npm run dev
   ```
   or
   
   ```bash
   npx vite
   ```
   
5. **Open the local URL shown in your terminal**
