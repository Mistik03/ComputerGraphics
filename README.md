This repository contains my practical assignments and projects for **Computer Graphics**.  
Each assignment is organized in its own folder.

## How to Run

### Prerequisites
- [Node.js](https://nodejs.org/) (comes with npm)

### Steps
1. Clone this repository:
   ```bash
   git clone https://github.com/Mistik03/ComputerGraphics.git

2. Navigate into the project folder you want to run. Example:
   ```bash
   cd .../ComputerGraphics

3. Install all required dependencies (this will automatically install Vite, Three.js, and any other libraries the project needs):
   ```bash
   npm install

4. Start the server:
   ```bash
   npm run dev
   ```
   
   or
   ```
   npx vite
