Each project and assignment is organized in its own folder.

## How to run

### Prerequisites
- [Node.js](https://nodejs.org/) (includes npm)

### Steps
1. **Clone this repository:**</br>

If your Git doesn't already support LFS, install it first:
```
git lfs install
```
then
```
git clone https://github.com/Mistik03/ComputerGraphics.git
```
2. **Navigate into the practical you want to run, for example:**
```
cd ComputerGraphics/PracticalFour
```

3. **Install dependencies:**
```
npm install
```
&
```
npm install three
```

4. **Start the dev server:**
```bash
npm run dev
```
